## 🧪 How to test by yourself

Copy the project 

```bash
  $ git clone https://github.com/GuilhermeHVSantos/nlw-return-impulse-web.git
```
    
Install the dependecies  

```bash
  $ npm install
```

Start the application 

```bash
  $ npm run start
```
